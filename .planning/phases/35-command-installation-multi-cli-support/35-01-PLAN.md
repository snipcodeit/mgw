  <name>Task 1: Add provider detection and --provider flag to mgw-install.cjs</name>
  <files>bin/mgw-install.cjs</files>
  <action>
Rewrite bin/mgw-install.cjs to support multi-CLI install. Keep the existing copyDirRecursive() helper unchanged. Add the following:

1. Add `const { execSync } = require('child_process');` to requires.

2. Define PROVIDER_TARGETS map:
   ```js
   const PROVIDER_TARGETS = {
     claude:   path.join(os.homedir(), '.claude', 'commands', 'mgw'),
     gemini:   path.join(os.homedir(), '.gemini', 'commands', 'mgw'),
     opencode: path.join(os.homedir(), '.opencode', 'commands'),
   };
   ```

3. Parse --provider flag from process.argv (handle both `--provider=claude` and `--provider claude`).
   If flag is present but value is not one of claude/gemini/opencode: print error listing valid options, exit 1.

4. Add detectProvider(forcedProvider) function:
   - If forcedProvider is set, return it (skip binary checks).
   - Otherwise try binaries in priority order: claude, gemini, opencode.
   - For each: call execSync('<binary> --version', { encoding: 'utf-8', stdio: ['pipe','pipe','pipe'] }).
   - Return the first that succeeds without throwing. If all fail, return null.

5. No-CLI guard: if detectProvider() returns null, print:
   `'mgw: no AI CLI found (claude/gemini/opencode) — skipping slash command install'`
   and exit 0.

6. Old-dir cleanup: read ~/.mgw-install-state.json. If it exists, parse it and compare `state.provider` to the detected provider. If they differ, remove `PROVIDER_TARGETS[state.provider]` with `fs.rmSync(oldDir, { recursive: true, force: true })`.

7. Determine targetDir = PROVIDER_TARGETS[detectedProvider].
   Parent dir guard: if `path.dirname(targetDir)` does not exist, print:
   `'mgw: ~/.{provider}/ not found — skipping slash command install (run the CLI once to initialize it)'`
   and exit 0.

8. Run the install: `const fileCount = copyDirRecursive(sourceDir, targetDir);`

9. Write state file: `fs.writeFileSync(statePath, JSON.stringify({ provider: detectedProvider }, null, 2));`

10. Output: `console.log('mgw: detected ' + detectedProvider + ' CLI — installed ' + fileCount + ' slash commands to ' + targetDir);`

Remove the old hardcoded claudeDir and targetDir constants. Update the file header comment to describe multi-CLI behavior.
